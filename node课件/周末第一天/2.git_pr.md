## 作业提交
- 讲师-> 组长 ->组员

## 讲师
- 建立一个作业仓库
- 克隆线上代码到本地
可以将线上的代码直接复制到本地，并且会默认下载master分支，给我们origin默认的远程地址

## 组长负责fork老师的仓库
- 复制（那一时刻的代码）一份 ，放到自己的仓库下，老师代码更改不会导致自己的代码发生变化
- 建立老师为远程仓库
```
git remote add teacher 地址
git pull //拉取老师的最新代码
```

- 给组员开通操作自己仓库的权限
